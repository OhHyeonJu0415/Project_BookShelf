## 기타 추가 사항
React로 프로젝트를 진행하면서 웹 컴포넌트 설계와 중복되는 코드의 양이 많아지는 것을 느꼈고, 좀 더 탄탄한 기본기가 필요하다고 판단하였습니다.

따라서, 기본기를 쌓기 위해 Vanilla JS/CSS를 사용하는 프로젝트들을 먼저 진행하기로 하고 해당 프로젝트는 잠시 중단하였습니다.
